### 一次一 IP
- 定义：本项目的核心效果描述：每次新建出站连接时从给定 IPv6 前缀中随机挑选一个地址作为 LocalAddr，使得目标服务器看到的来源 IP 各不相同，等价于拥有海量出口 IP 的代理池。

### /112 子网隔离
- 定义：部署策略：在同一个 /64 分配下，每台服务器只使用其中一段 /112 子网，避免多台机器同时绑定同一前缀导致冲突，实现横向扩展时的互不干扰。

### 本地路由绑定
- 定义：Linux 内核能力：通过 `ip route add local <prefix>/112 dev <iface>` 将指定 /112 子网路由到本地环回，再结合 `net.ipv6.ip_nonlocal_bind=1`，允许程序用该子网内任意未实际分配的 IPv6 地址作为套接字源地址发起 tcp6 连接。
