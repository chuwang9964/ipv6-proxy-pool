- 对外暴露的 CLI 参数统一通过文件级 `flag.String`/`flag.Int`/`flag.Bool` 定义于 `var` 块，并在 `main` 开头集中 `flag.Parse()` 与默认值设置。
- 并发限制采用带容量的缓冲 channel（`sem := make(chan struct{}, *limit)`）配合非阻塞 `select` 实现拒绝式背压，成功获取后 `defer` 释放。
- TCP 双向转发复用同一模板：起两个 `io.Copy` goroutine，分别关闭远端与本地连接，并用 `sync.WaitGroup` 等待双方结束。
- 错误路径统一通过 `log.Printf` 打印 `[HTTP-FAIL]` / `[SOCKS5-FAIL]` / `[HTTP-OK]` / `[SOCKS5-OK]` 等带标签的日志，便于区分协议与结果。
- 日志器在 `init()` 中按级别初始化四个 `log.Logger`（INFO/WARN/ERROR/DEBUG），并通过 `debugf` 仅在 `-v` 开启时输出调试信息。
