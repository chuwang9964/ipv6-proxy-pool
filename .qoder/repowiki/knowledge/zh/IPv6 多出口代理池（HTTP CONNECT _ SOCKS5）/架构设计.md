单一 `main` 包入口，无第三方依赖（`go.mod` 仅声明 module 与 go 版本），所有逻辑集中在 `main.go`：
- 启动阶段通过文件级 `flag.String/Int/Bool` 变量解析 `-http`、`-socks`、`-prefix`、`-c`、`-v`，校验 CIDR 后构造共享的 `server` 结构体（持有 `net.IPNet`、`*rand.Rand`、互斥锁与容量为 `limit` 的缓冲 channel 作为全局并发信号量）。
- 并行启动两个监听 goroutine：HTTP 使用自定义 `http.Server` + `http.HandlerFunc(s.handleHTTP)` 绕过 `DefaultServeMux`，仅接受 `CONNECT` 方法并通过 `http.Hijacker` 劫持 TCP；SOCKS5 使用裸 `net.Listen("tcp")` 循环 accept 后分派 goroutine 处理握手/请求/转发。
- 双向数据转发统一采用 `io.Copy` 双 goroutine + `sync.WaitGroup` 模式，并在 `Dialer.LocalAddr` 中注入 `randomIP()` 生成的随机 IPv6 源地址，实现“多出口 IP 池”效果。
- 运维脚本位于 `scripts/`：`install.sh` 负责在 Ubuntu/Debian 上拉源码、编译、写入 systemd 服务与内核参数；`gen_cert.sh` 生成 RSA 自签证书供 xray/v2ray TLS 使用。
